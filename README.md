# Log Aggregator Service

A Spring Boot 3 application with Spring Batch for log aggregation.

## Requirements

- Java 21
- Maven 3.6+

## Building the Project

```bash
mvn clean install
```

## Running the Application

```bash
mvn spring-boot:run
```

The application will start on port 8080.

## Technologies Used

- Spring Boot 3.2.3
- Spring Batch
- Java 21
- H2 Database
- Maven
